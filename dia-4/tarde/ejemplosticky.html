Mi prueba de sticky
Soy Sticky y ¡No me muevo!

Este es el contenido de la página. Al desplazarse se puede ver cómo el valor sticky se mantiene visible mientras se hace scroll.

Para que sticky funcione, el elemento tiene que estar dentro de un contenedor (el padre) que también permita el desplazamiento.

Si no hay desplazamiento o el elemento es el único hijo del contenedor, no funcionará.
